# 2026-07-10 그래프 검색 단순화 구현 — LLM 플래너 제거, 임베딩 시딩 단일 경로

## 배경

이전 세션 노트 `2026-07-10_search-context-분석-그래프검색-단순화-준비.md` 의
"권장 베이스라인 — 최소 버전"을 그대로 구현. 결정 기록은 D-047.

```
쿼리 임베딩 → search_entities_by_embedding(threshold=0.5, top_k=5)
→ 각 시드 1-hop 확장 → 노드 집합 내부 엣지 수집
→ 텍스트 포맷 + document_ids
```

## 구현 내용

### 신규: `src/context_loop/processor/graph_search.py`

- `search_graph(query_embedding, graph_store, *, embedding_client=None,
  seed_threshold=0.5, seed_top_k=5) -> GraphSearchResult | None`
- LLM 호출 0회, fallback 0층, 결정적. 게이팅은 "threshold 를 넘는 시드
  없으면 None (그래프 섹션 생략)" 으로 should_search LLM 판단을 대체.
- 엔티티 임베딩 lazy 보완(`build_entity_embeddings`)을 함수 안으로 흡수.
- entities 순서: 시드(유사도 내림차순) 먼저 → 1-hop 이웃. rank 메트릭
  (MRR/NDCG) 의 rank-1 이 쿼리 최유사 노드.
- 호출부 계약 유지: `GraphSearchResult(text, document_ids, entities,
  relations)`, description 채움(`_natural_description` — 평가 T4),
  `GraphRelationRef` 노출(`--score-relations`).

### 삭제: `src/context_loop/processor/graph_search_planner.py`

`plan_graph_search` / `execute_graph_search` / `GraphSearchPlan` /
`SearchStep` / `TargetEntity` / `TargetRelation` / `_render_system_prompt` /
`_build_seed_embeddings` / 3층 fallback / priority 정렬 / focus_relations
필터 전부 제거. 테스트 `test_graph_search_planner.py` 도 삭제.

### 수정

- `mcp/context_assembler.py`: `_search_graph_with_llm` 제거,
  `_maybe_graph` 의 `llm_client` 가드 제거 (그래프 탐색이 LLM 없이 동작).
  `llm_client` 는 HyDE 전용으로 역할 축소. `graph_planner_seed` 파라미터
  제거 (assemble_context / assemble_context_with_sources /
  _rerank_and_search_graph).
- `scripts/eval_search.py`: `--planner-seed-base` 플래그·배관 제거,

  그래프 검색은 구조적으로 결정적" 으로 갱신.
- `processor/llm_body_extractor.py`: 플래너 언급 주석 정리.

## 남은 검토 포인트 (후속)

- threshold 0.5 / top_k 5 실측 조정 — 짧은 엔티티 이름 vs 문장형 쿼리의
  임베딩 비대칭으로 그래프 섹션 생략 빈도 확인 필요 (골드셋 평가 재실행).
- `graph_store.py` 의 스키마 포맷 메서드 4종(get_schema_summary /
  format_schema_for_llm / get_query_relevant_schema /
  format_query_relevant_schema_for_llm)은 사용처가 사라졌지만 API 로 보존
  — 웹 UI 활용이 없다면 후속 정리 후보.
- `graph_vocabulary` 의 intent 매핑/프롬프트 포맷 함수도 인덱싱 외 사용처
  없음 — 동일하게 후속 정리 후보.
